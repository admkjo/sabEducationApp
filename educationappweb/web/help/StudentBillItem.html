Student Bill Item:

This is where the administrator gets to add all the various bill items applicable in the school. You can also edit or delete items already added.

To add, type in the item name and its description and click on the ‘save’ button. To edit on an entry, search using any of the search parameters and click on ‘select’ from its far right column. You can then edit the details of the item. Clicking on the ‘edit’ button saves the edited copy whiles clicking on ‘delete’ removes the Bill item from the list.
