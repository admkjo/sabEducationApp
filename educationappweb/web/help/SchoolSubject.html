School Subject:

This page allows you to enter the various subjects under each program being offered in the school. You can also edit or delete the entries already saved.

To add a subject, enter the subject code, initials and name and choose if it is a core subject or an elective subject. Clicking on ‘save’ adds it to the school subject list.

To edit on saved subjects, search by using any of the search parameters and click on ‘select’. You can then edit it and click on ‘edit’ to save the edited information. Clicking on ‘delete’ removes the subject selected from the school subject list.
