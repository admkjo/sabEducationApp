Academic Year Active Classes:

On this page, you can choose which classes are going to be active for the academic year and also remove classes that are going to be inactive for the academic year.

      • To add a class to the academic year’s active class list;
  1. Choose the class from the school class section.
  2. Click on ‘save’.
  3. Click ‘Reload Current Term Active Academic Classes’ to reload all the saved active classes so you can confirm if the new ones have been added.
      • To delete some classes;
  1. Search for the class by using either the academic year or school class.
  2. Click on ‘select’ from the far right of the class to be deleted.
  3. You can then click on ‘delete’ and confirm by clicking ‘ok’.
