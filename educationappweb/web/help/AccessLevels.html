ACCESS LEVELS

There is the Administrator level which is the highest and the administrator has access to all the features of the application and can choose which levels and features that should be available to all the other users of the application.

The various access levels that can be created by the Administrator include but not limited to the following;


Headmaster/mistress; This user may have control over all the functions just like the administrator as he/she is supposed to be the overseer of the software and its operations.


Non Teaching Staff; This user level may be assigned to the accounts department of the school so as to have access to the Student Finances section and the Audit tray.


Professional Teaching Staff; This user level is normally assigned to the professional teaching staffs of the school to give them access to the examination related section of the software.


Non Professional Teaching Staff; This user level is normally assigned to the non professional teaching staffs of the school and they also have access to the examination related section.


Lower Administrator; This user level is assigned some administrator privileges but does not have access to all the features of the software like the main administrator.


For the purposes of this manual we assume that administrator has log on.
