School Programme:

This is where you document the programs offered in the school. You can also edit or delete some of the data already entered.

To enter new programs, type in the program name and code in the spaces provided and click on ‘save’.

To edit, search using the search parameters and click on the ‘select’ button. You can then edit the data and click on ‘edit’ to save. You can also click ‘delete’ to remove the data from the system.
