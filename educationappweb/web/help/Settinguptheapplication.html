Setting up the application

Typically, different functions to be performed demands different data set being available before commencement.

However, before any activity can be performed, first and foremost the academic year and term should be available. This is because, every activity performed one way or the other belongs to a particular academic year and term.

To get things started,


1. You create the academic Year and academic term. They can both be set up under the ‘Year/Term Activities’ section.

2.  Select the academic term, it would appear in the current academic term button, and then click on it to set it as the current term.

3. Create programmes offered in the school i.e. General Science, Visual Art etc.

4.  Create the various subjects being taught in the school.

5.  Group subjects to form subject combination. (Subject combination is a group of school subjects that a particular student is examined on)

6.  Create the school classes.

7.  Create the various houses in the school. To create the school Programmes, subjects, Subject combination, classes and houses, refer to the ‘Set Up’ section.

8.  Then you can add the student information and load their pictures.

9.  Assign student their classes and subject combination at the class membership page under the ‘General Actions’ panel.
