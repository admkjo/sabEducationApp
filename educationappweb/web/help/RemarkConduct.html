Remark/Conduct:

As the administrator, you can choose all the available remarks and conducts of students to be used by the teachers in the preparation of reports for the students. You can also edit the comments you have already entered.

      • You can save the comments by typing them in the space provided for comment and choosing the type in which it should be classified be it remarks, conduct, attitude, interest or academic. You can click ‘save’ when done to effect the change.
      • To edit those you have already entered;
  1. Enter it in the search field and choose the classification by which to search for it, i.e., comment, type or by the school number.
  2. Clicking search afterwards will generate the options so you can choose the one to edit.
  3. Click on the ‘select’ on the tail end of the one you want to edit to load it in the edit field to be edited.
  4. After editing, click on ‘Edit’ to save the edited comment.
