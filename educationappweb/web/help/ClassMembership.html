Class Membership:

On this page, you can view the list of students in a class and also remove and add students from one class to the other. You can also generate a class membership list containing either their full details or to be used as a class sign list or as an examination sheet for each subject.

      • To view the class membership list, you have to choose the year group, programme and class and click on ‘View Class Membership’.
      • To remove or add a student to a particular class;
  1. Enter the student’s ID in the space provided.
  2. Choose the subject combination in which to remove or add the student.
  3. You then choose if to remove or add the student from the subject combination chosen
  4. Click on ‘Execute’ to complete.
      • To generate a report;
  1. Type in what you want the name of the report to be.
  2. Choose the type of report you want, i.e., student full detail, class sign list or examination sheet.
  3. You can then choose the subject/course if it is an examination sheet you want.
      • Note that you still have to choose the year group, programme and class for which you want to generate the report for before clicking the ‘Generate Report’ button.
